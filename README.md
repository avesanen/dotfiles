Dotfiles
========
My dotfiles for linux and mac.

Arch (btw)
----------
```
sudo pacman -S nerd-fonts neofetch alacritty
git clone git@github.com:avesanen/dotfiles.git
cd dotfiles
cp -r .* ~
```

OSX
---

```
brew install --cask font-0xproto-nerd-font
git clone git@github.com:avesanen/dotfiles.git
cd dotfiles
cp -r .* ~
```
